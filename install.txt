== Description ==

This plugin implements cryptocurrency support for Easy Digital Downloads.
== Installation ==

1. Upload files to the `/wp-content/plugins/easy-digital-downloads-coinpayments-currencies` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Easy Digital Downloads Settings page on General tab go to "Currency" section tab.
4. Select any cryptocurrency from Coinpayments.NET list.
5. Click "Save changes".
